# Experiment data directory

Store raw probe captures, scoring sheets, HTML/schema snapshots, and schedule
logs here: one subdirectory per experiment:

```text
data/exp-001/raw/
data/exp-001/scores/
data/exp-002/...
```

Do not commit secrets, private client data, or account credentials.
Prefer anonymized client references when publication requires it.

This folder intentionally starts empty: no fabricated result files.
